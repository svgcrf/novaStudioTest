Child theme de Hello Elementor optimizado para landing pages de alta conversión.

== Description ==

Nova Studio es un tema hijo de Hello Elementor diseñado específicamente para crear landing pages profesionales y de alta conversión. Incluye:

* Sistema de diseño completo con variables CSS
* Tipografías optimizadas (Plus Jakarta Sans + Inter)
* Paleta de colores profesional
* Componentes pre-diseñados
* Optimizaciones de rendimiento
* 100% compatible con Elementor

== Installation ==

1. Asegúrate de tener instalado el tema Hello Elementor
2. Sube la carpeta `nova-studio` a `/wp-content/themes/`
3. Activa el tema desde Apariencia > Temas
